# Zenith-AI
Zenith AI is a terminal-based security auditor designed for deep "hole hunting." It crawls specific project folders to locate logic flaws, leaked secrets, and vulnerabilities. Featuring a sleek, interactive dashboard, Zenith provides hackers and developers with instant exploit explanations to identify critical weaknesses.

How To Use It?
To use Zenith AI, follow these steps to initialize the auditor and begin your security scan
(OPEN THE BATCH FILE CALLED- zenith.bat)

1. Provide Authentication
The terminal will prompt you with > Paste Gemini API Key:. Paste your unique key here; the AI is instructed to store this for the session without repeating it in the chat to keep your credentials secure.

2. Select the Target Folder
Once authenticated, the AI will ask for the > Folder to review:. Type the name of the directory you want to inspect (e.g., Code or src). The AI will then crawl your workspace to locate this folder everywhere on your system.
